#_itrator_#
An iterator is an object that is used to point to and move through elements of a container

1. Input Iterator : it is used to read elements from a container one by one in forward direction
[https://github.com/RutviParakhiya2007/CGxSU_Semester_1/blob/main/cpp_language(sem_02)/23.stl_iterators.md#1-input-iterators]

2. Output Iterator : it is used to write elements to a container one by one
[https://github.com/RutviParakhiya2007/CGxSU_Semester_1/blob/main/cpp_language(sem_02)/23.stl_iterators.md#2-output-iterators]

3. Forward Iterator : it is used to read or write elements while moving only forward in a container
[https://github.com/RutviParakhiya2007/CGxSU_Semester_1/blob/main/cpp_language(sem_02)/23.stl_iterators.md#3-forward-iterators]

4. Bidirectional Iterator : it can move forward and backward through elements of a container
[https://github.com/RutviParakhiya2007/CGxSU_Semester_1/blob/main/cpp_language(sem_02)/23.stl_iterators.md#4-bidirectional-iterators] 

5. Random Access Iterator : can directly access any element in a container using an index
[https://github.com/RutviParakhiya2007/CGxSU_Semester_1/blob/main/cpp_language(sem_02)/23.stl_iterators.md#5-random-access-iterators]
